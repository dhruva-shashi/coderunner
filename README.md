# Coderunner

## Overview

Coderunner is a tool which is used to compile C++ files and run the executable file. The input is read from an input file and the output is redirected to an output file. 

## Installation

To install coderunner, click [here](https://github.com/dhruva-shashi/coderunner/raw/master/installer) and perform the following steps.

### Give permissions

- Give permission for the installer to install Coderunner by using the following command

`chmod +x installer`

- Run the installer executable file

`./installer`

## How to Use

- Create a file for input and name it `input.txt`
- Create a file for output and name it `output.txt`
- Write the code to a file named `codefile.cpp`
- Run coderunner using the `$ coderunner` command
